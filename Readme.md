- プログラムに関して
- 使用画像ファイルなど: file.zipにあります．
- Python 3.13.5で動作確認
- 必要ライブラリ（pip3でインストールする）
  - opencv-python
  - opencv-contrib-python
  - jupyterlab
  - numpy
  - matplotlib
  - scikit-learn
- プログラムを実行する場合はプログラムと使用画像ファイルと同じフォルダにすること
  - プログラムとファイルの構造関係分かっている場合，ファイルパスを適切に指定すれば動作可能
